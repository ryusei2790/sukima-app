# 隙間時間ルーレット 使い方

## アプリの概要

隙間時間に「次にやること」をランダムに選ぶシンプルなWebアプリです。
考えるより先に体を動かす、そのきっかけをつくります。

## 使い方

### Step 1: アプリを開く

ブラウザでアプリにアクセスします。
「ボタンを押して抽選」と表示されたカードと「回す」ボタンが見えます。

### Step 2: 「回す」ボタンを押す

ボタンをタップ／クリックすると、抽選が始まります。
スピナーが約1.5秒間回転します。

### Step 3: 結果を確認する

カードにタスク名が大きく表示されます。
表示されたタスクをそのままやってみましょう。

### Step 4: もう一度引きたい場合

「もう一度」ボタンで最初の状態に戻ります。

## 現在のタスク一覧

- 本を読む
- 深呼吸する
- 瞑想する

## カスタマイズ方法

`src/data/tasks.ts` のタスク配列を編集することで、自分好みのタスクに変更できます。

```typescript
export const tasks: Task[] = [
  { id: "read", label: "本を読む" },
  { id: "breath", label: "深呼吸する" },
  { id: "meditate", label: "瞑想する" },
  // ここに追加する
  { id: "walk", label: "散歩する" },
];
```

## 技術スタック

- Next.js 16 / React 19 / TypeScript
- Tailwind CSS v4

## セットアップ

```bash
npm install
npm run dev    # 開発サーバー起動（http://localhost:3000）
npm run build  # 本番ビルド
npm start      # 本番サーバー起動
```
